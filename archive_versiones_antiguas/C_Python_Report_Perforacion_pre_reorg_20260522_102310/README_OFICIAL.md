# Sistema de Reporte de Perforación

Carpeta oficial:

`C:\Python_Proyectos\Report_Perforacion`

Versión actual:

`v1.0.5 - Dashboard KPI Profesional`

Para ejecutar:

```powershell
cd C:\Python_Proyectos\Report_Perforacion
python -m streamlit run app_perforacion.py
```

Antes de modificaciones importantes, crear respaldo:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\backup_project.ps1
```

No ejecutar copias antiguas ubicadas en Desktop o carpetas de respaldo.

Preparar Git local cuando Git para Windows esté instalado:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\init_git_local.ps1
```
